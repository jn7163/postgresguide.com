安装 Postgres
===================

Mac OSX 用户
~~~~~~~~~~~~~~

- 通过适用于Mac OSX的App `Postgres App <http://www.postgresapp.com>`_
- 通过Homebrew编译安装

.. code-block:: sh

   brew install postgresql

Ubuntu 用户
~~~~~~~~~~~~

.. code-block:: sh

   sudo apt-get install postgresql

Windows 用户
~~~~~~~~~~~~

`Windows 安装包 <http://www.enterprisedb.com/products-services-training/pgdownload#windows>`_
